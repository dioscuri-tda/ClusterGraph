Can be “max” or “min”. When the parameter “type_coloring” is set to “label”, it defines the way to choose the label inside each node to color them. If “max” is chosen, the most represented label inside each node will be chosen. If “min” is chosen it will be the least represented label. , by default “max”

coloring_strategy_varstr, optional

Can be “log”, “lin” or “exp”. When the parameter “type_coloring” is set to “variable”, this parameter represents how fast color will changed between nodes depending on the variable’s average value inside each node. For example if “exp” is chosen, a slight change of average value between two nodes will represent an important change of colors. , by default ‘lin’

MIN_SIZE_NODEfloat, optional

The minimum size of nodes in the plot, by default 0.1
